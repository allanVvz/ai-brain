# Memory: Patterns

## Template

```json
{
  "id": "",
  "signal": "",
  "context": "",
  "action": "",
  "result": "",
  "date": ""
}
```

## Usage

- Add a new entry each time a pattern is confirmed
- `signal`: what triggered the pattern
- `action`: what was done
- `result`: outcome (success / failure)
