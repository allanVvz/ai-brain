# Baita Conveniência — Formatos que Funcionam

## A) Carrossel

### Dados
- **Engajamento estimado:** 3-8% (maior que post estático)
- **Slides ideais:** 3-5 (máximo 7)
- **Motivo:** Instagram empurra mais para quem interage; cada swipe é uma interação

### Estrutura Ideal para Bar

| Slide | Função | Exemplo |
|-------|--------|---------|
| 1 | Gancho (stop scroll) | "Por que você ainda não foi no Baita essa semana?" |
| 2 | Desenvolvimento / humor | Situação relatable |
| 3 | Punch line ou oferta | Produto + preço ou meme |
| 4 (opt) | Prova social | Foto real do lugar / avaliação |
| Último | CTA leve | "A cerveja tá aqui. Você sabe o endereço." |

### Exemplos de Carrossel que Convertem

1. **"5 desculpas que você usa pra não sair"** (humor ácido) → último slide: "O Baita resolve todas."
2. **Antes/depois** (segunda vs sexta no humor) → último slide: oferta do final de semana
3. **Produto do dia** (3 slides: produto, preço, como pedir) → CTA no WhatsApp

## B) Reels (Instagram) / TikTok

### Dados
- **Alcance:** 3-10x maior que post estático para contas pequenas
- **Engajamento médio:** 5-20% (quando acerta)
- **Duração ideal:** 7-15 segundos (máximo 30s para bar)
- **Primeiros 2 segundos:** decisivos — prender ou perder

### Estrutura de Reel para Bar

```
0-2s:  Gancho visual ou texto forte
2-8s:  Desenvolvimento (humor, produto, situação)
8-15s: Punch line ou CTA
```

### Tipos de Reel que Funcionam para Bar

| Tipo | Descrição | Engajamento |
|------|-----------|-------------|
| POV no balcão | "POV: você entrou pra comprar água" | Alto |
| Timelapse da noite | Bar vazio → cheio | Médio-alto |
| Produto em close | Cerveja gelada sendo aberta | Alto (sensorial) |
| Texto animado | Meme estático mas com movimento | Médio |
| Trend de áudio | Adaptar trend do momento ao bar | Alto se timing certo |

### Erros Comuns

- Reel muito longo (> 30s para bar = morte)
- Texto que explica demais
- Qualidade de vídeo ruim sem intenção (ruim intencional funciona; acidentalmente ruim não)
- Não usar legenda/texto sobreposto

## C) Stories

### Dados
- **Alcance:** menor que feed, maior retenção
- **Engajamento:** resposta + enquete + sticker
- **Frequência ideal:** 2-4 stories/dia (não mais)

### Tipos de Stories por Objetivo

| Objetivo | Tipo | Exemplo |
|----------|------|---------|
| Engajamento | Enquete / pergunta | "Qual cerveja hoje? A ou B?" |
| Avaliação | CTA direto | "Avalia no Google, ganha drink" |
| Prova social | Repost de menção | Cliente marcou o bar → repostar |
| Bastidor | Foto real do lugar | Geladeira abastecida, balcão, equipe |
| Promoção | Imagem simples + texto | "Hoje: [produto] R$X até 22h" |

### Story de Alta Conversão (campanha avaliação)

```
Slide 1: "Fez algo bom essa semana?"
Slide 2: Enquete — Sim / Não
Slide 3: "Independente da resposta: avalia o Baita no Google e ganha um drink."
Slide 4: [Print do Google Maps com 5 estrelas]
Slide 5: CTA — "Link na bio ou aparece aqui"
```

## D) Post Estático

### Dados
- **Engajamento:** mais baixo que carrossel/reel
- **Uso ideal:** produto visual forte, meme simples, timing (data especial)
- **Quando usar:** quando a imagem é autoexplicativa

### Post Estático que Funciona para Bar

- Foto real do produto (não stock photo)
- Meme com texto apenas (sem foto de produto)
- Horário/info operacional (sempre performa bem — utilidade)
- Avaliação real de cliente (print + moldura)

## E) Métricas Benchmark

### Taxa de Engajamento por Formato (bares Brasil — estimativa de mercado)

| Formato | Micro (<5k) | Médio (5-50k) | Grande (>50k) |
|---------|-------------|---------------|---------------|
| Post estático | 3-8% | 1-4% | 0,5-2% |
| Carrossel | 5-12% | 2-6% | 1-3% |
| Reel | 8-25% | 3-12% | 1-5% |
| Stories (view rate) | 20-40% | 15-25% | 8-15% |

> **Para Baita (conta nova):** esperar resultados do lado "micro" até 3 meses de consistência.

### Melhores Horários para Publicar (Novo Hamburgo / RS)

| Dia | Horário | Justificativa |
|-----|---------|--------------|
| Segunda-Quarta | 12h-13h / 18h-19h | Almoço e saída do trabalho |
| Quinta | 17h-19h | Pré-rolê de quinta |
| Sexta | 12h-13h / 17h-20h | Maior pico da semana |
| Sábado | 14h-17h | Público ativo no período da tarde |
| Domingo | 18h-20h | Domingo à noite / "última chance da semana" |

| Fase | KPI Principal | Meta |

| Mês 1-2 | Seguidores | +200/mês |

| Mês 3+ | Avaliações Google | +40% (objetivo declarado) |
| Contínuo | Mensagens/DMs | Resposta em < 2h |
