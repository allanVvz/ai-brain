# Baita Conveniência — Copys que Funcionam

## Estrutura de Copy que Funciona em Bar

### Fórmula base

```
[Gatilho emocional / situação relatable]
[Punch line / humor / virada]
[CTA leve ou implícito]
```

### Abertura que para o scroll

- Pergunta provocativa: "Você ainda tá no trabalho, né?"
- Afirmação ácida: "Ninguém merece segunda-feira."
- Situação específica: "POV: você disse 'só uma' às 18h."
- Timing: "Sexta-feira, 17h59. O que você tá fazendo aqui?"

### CTA que não parece CTA

✓ "Você sabe onde a gente tá."
✓ "A cerveja tá aqui. O resto é contigo."
✓ "Se aparecer, a culpa é sua." (irônico)
✗ "Clique no link da bio para mais informações."
✗ "Venha nos visitar!"

## Análise de Copys Reais que Funcionam em Bares BR

> ⚠️ Copys abaixo baseadas em padrões documentados de alta performance em bares brasileiros e marcas com tom similar. Textos exatos de perfis específicos requerem validação direta no Instagram.

### Copy 1 — Timing perfeito (sexta)
```
"Meu chefe perguntou se eu ficaria até mais tarde hoje.
Eu disse que ia pensar.
Pensei. Não."

[Foto: balcão com cerveja gelada]
```
**Por que funciona:** timing (sexta tarde), identificação imediata, humor sem piada forçada.
**Engajamento estimado:** 8-15% (salvamentos + compartilhamentos altos)

### Copy 2 — Expectativa vs Realidade
```
"Segunda-feira você: 'semana que vem eu não saio'
Sexta-feira você: [foto do lugar cheio]"
```
**Por que funciona:** carrossel de 2 slides, humor relatable universal, sem texto desnecessário.
**Engajamento estimado:** 6-12%

### Copy 3 — Honestidade brutal (antimarketing)
```
"A gente não tem o drink mais instagramável da cidade.
A gente tem o preço que você consegue pagar sem repensar as escolhas de vida."
```
**Por que funciona:** quebra expectativa, honestidade que vira piada, posicionamento por contraste.
**Engajamento estimado:** 10-20% (compartilhamentos e comentários de identificação)

### Copy 4 — Bilhete (baixíssima produção, alto retorno)
```
[Foto de bilhete escrito à mão:]
"Se você tá lendo isso numa quinta, já tá quase lá.
— O Baita"
```
**Por que funciona:** autenticidade visual, sem produção, sensação de mensagem pessoal.
**Engajamento estimado:** 5-12%

### Copy 5 — Campanha "Avalia e Ganha Drink"
```
"Sabe quando você faz uma coisa boa e não ganha nada?
Pois é. Aqui não.
Avalia o Baita no Google, mostra pra gente e ganha um drink.
Simples assim, tchê."
```
**Por que funciona:** CTA claro sem parecer propaganda, tom honesto, recompensa imediata.
**Engajamento estimado:** 4-8% + conversão em avaliações

## 20 Variações de Copy para Baita Conveniência

### Tom: Suave (para alcançar público novo)

**S1 — Quinta promissora**
```
Amanhã é sexta.
Isso não precisa de mais explicação.
📍 Baita Conveniência, Novo Hamburgo.
```

**S2 — Produto simples**
```
Cerveja gelada.
Preço justo.
Sem frescura.
Nos vemos mais tarde, tchê.
```

**S3 — Convite sem pressão**
```
Se a semana foi boa: comemora aqui.
Se a semana foi ruim: passa aqui.
A gente tá sempre aberto.
```

---

### Tom: Normal (padrão de conteúdo)

**N1 — POV clássico**
```
POV: você disse "só vou passar rápido" no Baita.

[próximo slide: 3h depois]
```

**N2 — Monday mood**
```
Segunda-feira humor: 💀
Sexta-feira humor: 📈

O que mudou? Nada.
A cerveja já tá gelada aqui, bah.
```

**N3 — Timing de jogo**
```
Hoje tem jogo.
A cerveja tá gelada.
O resto é problema do time.
```

**N4 — Avalia e Ganha**
```
Fez algo bom essa semana?
Não? Tudo bem.
Avalia o Baita no Google, mostra aqui e ganha um drink.
Você merece uma recompensa mesmo assim, tchê.
```

**N5 — Honestidade de produto**
```
Não temos o drink mais caro da cidade.
Temos o que você vai querer de novo amanhã.
```

**N6 — Situação relatable**
```
Você: "não vou sair hoje"
Você às 20h:
[foto do balcão do Baita]
```

**N7 — Resposta ao comentário como post**
```
Alguém perguntou: "vocês fecham que horas?"
Resposta: quando a última pessoa for embora.
(Ou às 2h. Depende da noite.)
```

---

### Tom: Ousado (para engajamento e compartilhamento)

**O1 — Dark humor relatable**
```
Terça-feira é o dia mais inútil da semana.
Mas pelo menos a cerveja funciona.
```

**O2 — Antimarketing direto**
```
Não segue nosso Instagram porque gosta de nós.
Segue porque você sabe que vai aparecer aqui em algum momento.
Relaxa. A gente sabe disso.
```

**O3 — Verdade inconveniente**
```
Você não "vai só passar pra pegar uma coisa".
Nunca foi assim. Nunca vai ser.
Tá tudo bem, bah.
```

**O4 — Sarcasmo com afeto**
```
Obrigado por aparecer aqui toda sexta e dizer "que saudade".
A gente não foi a lugar nenhum.
Vocês que sumom.
```

**O5 — Meme de preço**
```
O delivery cobra R$ 8 de entrega em 3km.
Aqui você vem de graça e a cerveja já sai mais barata.
Só dizendo, tchê.
```

**O6 — Dark humor + CTA**
```
Sua semana foi osso?
A nossa também, não vou mentir.
Mas a cerveja tá gelada e o preço não mudou.
Aparece.
```

**O7 — Marca amigo (viralização)**
```
Marca aquele amigo que você quer levar no Baita
mas ele sempre fala que não pode.

(esse post é pra ver se ele vê)
```

## Estrutura de Copy por Formato

| Formato | Abertura | Corpo | CTA |
|---------|----------|-------|-----|
| Feed estático | Situação relatable | Punch line | Implícito ou irônico |
| Carrossel | Slide 1: gancho forte | Slides 2-4: desenvolvimento | Último slide: CTA leve |
| Reel | Primeiros 2s decisivos | Humor visual + texto mínimo | Nenhum ou 1 linha |
| Stories | Pergunta ou enquete | — | Swipe up / resposta |
| WhatsApp | Direto ao ponto | Max 5 linhas | CTA claro e simples |
