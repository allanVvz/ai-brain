# Baita Conveniência — Estratégia WhatsApp

## A) Estrutura de Grupos

### Configuração Recomendada

| Grupo | Tamanho | Propósito |
|-------|---------|-----------|
| Baita VIPs 🍺 | Até 150 pessoas | Clientes frequentes, promoções exclusivas |
| Baita Novidades | Até 250 pessoas | Anúncios gerais, cardápio, horários |
| Canal Baita (broadcast) | Ilimitado | Comunicação 1:1, sem interação em grupo |

> **Recomendação prática:** começar com **1 grupo principal + 1 canal**.
> Dividir por segmento só após 200+ contatos.

### Como Crescer Organicamente

- QR Code na balada/no caixa: "Entre no grupo e ganhe desconto"
- Story do Instagram direcionando para grupo
- "Avalia e Ganha" inclui convite pro grupo
- Funcionário convida pessoalmente clientes frequentes
- Post específico: "Quer saber das promoções antes de todo mundo?"

### Como Evitar Spam e Bloqueios

- Máximo **3 mensagens por semana** no grupo
- Nunca enviar entre 23h e 9h
- Sem correntes, sem conteúdo off-topic
- Admin responde sempre que alguém falar no grupo
- Clareza na entrada: "Aqui só tem promoção do Baita e nada mais, tchê"
- Opção de saída sempre disponível e sem drama

## B) Tipos de Conteúdo que Funcionam no WhatsApp

### Promoção do Dia (alta abertura)
```
🍺 PROMOÇÃO DE HOJE — Baita Conveniência

Hoje: [produto] por R$[X]
Válido até [hora] ou enquanto durar.

Aparece aqui, tchê!
📍 [Endereço]
```
**Frequência:** 1-2x/semana
**Melhor horário:** quinta ou sexta, 17h-18h30

### Aviso de Abertura/Novidade
```
Oi guria/guri 👋
Hoje tem [novidade/evento/produto novo].

Semana boa pra todo mundo.
A gente tá aqui se precisar.
```
**Frequência:** 1x/semana
**Tom:** leve, sem pressão

```

Qual você prefere amanhã?
1️⃣ Cerveja long neck gelada por R$X
2️⃣ Dose de [drink] por R$X

Responde aqui 👇
```
**Frequência:** 1x/quinzena
**Resultado:** usar para definir promoção do dia seguinte

```

Simples:
1. Abre o Google Maps
```

## C) Copy para WhatsApp — Regras

| Regra | Detalhes |
|-------|---------|
| Comprimento | 3-7 linhas (máximo) |
| Emojis | Sim, 1-3 por mensagem (pontual, não decorativo) |
| Gírias RS | Sim — bah, tchê, guri/guria, tri |
| Bullet point | Funciona para listas (promoções, passo a passo) |
| Formato texto corrido | Evitar — parece spam |
| Frequência | Máximo 3x/semana |
| Horário proibido | 23h-9h |

### Diferença Instagram x WhatsApp

| Instagram | WhatsApp |
|-----------|---------|
| Copy pode ter humor mais elaborado | Copy direto ao ponto |
| Imagem carrega a mensagem | Texto precisa funcionar sozinho |
| CTA implícito ok | CTA claro obrigatório |
| Pode ser mais longo | Máximo 7 linhas |
| Construção de brand | Conversão e relacionamento |

## D) Calendar — Timing

### Melhor Horário para Enviar

| Dia | Horário | Tipo de Mensagem |
|-----|---------|-----------------|
| Quinta | 17h-18h30 | Promoção de quinta/pré-sexta |
| Sexta | 12h-13h | Promoção de sexta à noite |
| Sábado | 14h-16h | Promoção happy hour/noite |
| Domingo | — | Evitar (resaca, mau humor 😅) |
| Segunda-Quarta | 17h-18h | Aviso leve, sem pressão |

### Frequência Ideal

- **3 mensagens/semana** = limite sem alienar
- **1-2 mensagens/semana** = ideal para fidelidade
- **5+ mensagens/semana** = risco alto de saída do grupo

## E) Gamificação

### Sistema de Pontos (simples, sem app)

**Como funciona:**
- Cada R$50 gastos = 1 "ponto" registrado no nome do cliente
- 5 pontos = 1 drink grátis
- Controle via planilha ou caderno no caixa

**Como comunicar no WhatsApp:**
```
💛 CLUBE BAITA — Como funciona:

A cada R$50 você ganha 1 ponto.
Juntou 5? Um drink é seu, tchê.

Não precisa de app, não precisa de nada.
Só aparecer e pedir que a gente sabe.
```

### VIP Simples

- Cliente que avaliou no Google = badge "Cliente VIP" no grupo
- Acesso antecipado às promoções (24h antes do post público)
- Sem custo para implementar

### Referral (Indica um Amigo)

```
Indicou um amigo pro grupo?
Se ele aparecer aqui e citar seu nome,
você ganha [desconto/drink].

Simples assim, bah.
```
**Estimativa de conversão:** 10-20% dos membros ativos indicam alguém se incentivado corretamente.
