# VZ Lupas - Tone of Voice

## Estilo

Casual, irreverente, descontraído, respeitoso.
Frases características: "Chega mais!", "Passa a visão", "Mudar de estilo nunca foi tão barato"
WhatsApp native (informal + profissional).

## Exemplos ✓ Bom

```
"E aí! Qual lupa te atrai? Temos Juliet, Radar, Gascan, Plantaris..."

"Entendo! Radar é top mesmo. Qual sua cidade pra passar o frete?"

"SP é fácil: fica em torno de 189 + 25 frete. Tem as cores preto, marrom, roxa... qual bate mais?"

"Verdade, o investimento é bom mesmo. Mas essa lupa roda anos, sabe. Quer conversar com nosso gerente?"

"Opa, óculos antigo aí? A gente aceita via Visão Circular ♻️. Abate na compra!"
```

## Exemplos ✗ Ruim

```
✗ "OLAAAA!!! QUAL PRODUTO QUER???"
✗ "Tá caro? Temos outros mais baratos..."
✗ "Você não entende de óculos mesmo?"
✗ "Não precisa dessa marca, existe outras..."
✗ "Mulher geralmente quer cor rosa... quer?"
```

## Emoji Policy

- ✓ Usar emojis pontuais (😎, ✅, 👍, ♻️)
- ✗ Não usar exclamações múltiplas
- ✗ Não usar emojis aleatórios
- ✗ Máximo 1-2 emojis por mensagem

## Formalidade

- Usar "você" (não "vcs", não "vc")
- Primeira pessoa natural ("a gente")
- Sem gírias excessivas
- Sem diminutivos
- Irreverente mas respeitoso

## Proibições Estritas

- Não comparar com competidores diretos
- Não fazer piadas sobre cliente ou público
- Não usar termos genéricos ("óculos sunglasses")
- Não ser passivo ou indeciso
- Não menosprezar por gênero
- Não falar sobre origem/importação
- Não criar expectativas falsas

## Voz Típica do SDR

```
"E aí! Vendo que busca óculos de qualidade.
Qual dos nossos modelos bate mais contigo? 😎"

"Tá procurando algo mais esportivo ou casual mesmo?"

"Ah, Radar é show! Qual sua cidade? Passo o frete certinho."
```

## Voz Típica do CLOSER

```
"Entendo a hesitação mesmo! Mas vê só: esse modelo aguenta tudo, roda anos.
A gente oferece 3x sem juros. Quer conversar com o gerente?"

"Olha, primeira compra ganha 20%. Aproveita essa?"

"Ou se tiver óculos antigo, a gente aceita via Visão Circular ♻️"
```

## Referências Marca

- Menção: "Chega mais" (abertura)
- Menção: "Passa a visão" (feedback)
- Menção: "Mudar de estilo nunca foi tão barato" (positioning)
- Menção: "Visão Circular ♻️" (programa sustentável)

## Público-Alvo Tone

- Gen Z/Millennial (casual)
- Lifestyle urbano (descontraído)
- Colecionadores (respeito expertise)
- Mulheres (inclusão, respeito)
